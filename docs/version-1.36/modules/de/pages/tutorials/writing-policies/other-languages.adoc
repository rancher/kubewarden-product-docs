{admc-name} Richtlinien können mit jeder Programmiersprache geschrieben werden, die WebAssembly-Module erzeugen kann, die auf die WASI-Schnittstelle abzielen.

Bitte öffnen Sie einen Pull Request oder ein GitHub-Issue https://github.com/kubewarden/docs[hier], wenn Sie von einer Programmiersprache wissen, die verwendet werden kann, um {admc-short-name} Richtlinien zu schreiben.
