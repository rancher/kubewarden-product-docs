== 策略测试角色

本节涵盖测试 {project-name} 策略的主题。
有两个角色对测试策略感兴趣：

* *策略开发者*：您正在编写 {short-project-name} 策略，并希望确保您的代码按预期运行。
* *集群操作员*：您发现了一个 {short-project-name} 策略，并希望在部署之前测试和调整策略设置。
您希望在 CI/CD 管道和其他管理任务中持续测试这些设置。

文档后续部分展示了这些角色如何测试 {short-project-name} 策略。
