.NET 内核目前对 WebAssembly WASI 平台提供实验性支持。
这来自于 https://github.com/SteveSandersonMS/dotnet-wasi-sdk[`dotnet-wasi-sdk`] 项目。

[NOTE]
====

您无需安装 Windows 即可编写或运行 .NET 内核代码。
所有操作也可以在 Linux 或 macOS 机器上完成。
====


== 当前状态

策略作者可以使用以下资源：

* https://github.com/kubewarden/policy-sdk-dotnet[{project-name} .NET 内核 SDK]：这提供了一组对象和函数，简化了编写策略的过程。
* https://github.com/kubewarden/policy-sdk-dotnet/tree/main/example[{short-project-name} 策略示例]：这是一个有效策略的示例。

已知没有限制。
SDK 使得编写验证和变更策略成为可能。

可以使用 https://www.nuget.org/packages/KubernetesClient.Models[`KubernetesClient.Models`] 库来处理 Kubernetes 对象。
