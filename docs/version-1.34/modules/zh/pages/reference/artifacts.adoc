== OCI 工件

对于开放容器倡议（OCI）工件，该项目将 `ghcr.io` 中的所有内容发布至 https://github.com/orgs/kubewarden/packages.。

- https://github.com/orgs/kubewarden/packages/policies[`kubewarden/packages/policies/*`] 是策略 Wasm OCI 工件。
- https://github.com/orgs/kubewarden/packages/charts[`kubewarden/packages/charts/*`] 是以 OCI 工件形式发布的 Helm 图表。它们遵循 SLSA 标准。
- 作为 OCI 镜像发布的容器镜像： `kubewarden-controller`， `policy-server`， `audit-scanner`， `kubectl`。
- 还有 `packages/tests/*` 包含用于端到端测试的工件。

== 通过 HTTPS 的 Helm 图表

{short-project-name} 将 Helm 图表发布到位于 https://charts.kubewarden.io. 的 Helm 图表 HTTPS 储存库中。它们没有 SLSA 或签名，但这种类型的 Helm 储存库提供搜索功能。
