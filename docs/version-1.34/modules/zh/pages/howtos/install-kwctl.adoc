= 安装 kwctl ({project-name} CLI)
:revdate: 2026-03-18
:page-revdate: {revdate}
:title: 安装 kwctl
:description: 按照各种操作系统的安装说明安装 kwctl（{project-name} CLI），然后启用外壳补全。
:keywords: ["{project-name}", kubewarden, kubernetes, 安装 kwctl, 安装, kwctl]
:doc-persona: kubewarden-all
:doc-type: HOWTO
:doc-topic: install-kwctl

`kwctl` 是 {project-name} 的命令行界面（CLI）工具。以下是一些操作系统的安装说明。

[tabs]
====
Linux::
+
--
在 Linux 上安装

. Homebrew
+
如果您在 Linux 上使用 Homebrew 包管理器，则：
+
[source,console]
----
brew install kwctl
----
+
验证安装：
+
[source,console]
----
kwctl --version
----

. Zypper（openSUSE）
+
[source,console]
----
zypper install kwctl
----

. AUR
+
如果您使用 Arch Linux 或基于 Arch 的发行版，可以从 AUR 安装 kwctl。
+
.. 使用 AUR 助手（yay）：
+
[source,console]
----
yay -S kwctl
----
+
.. 使用 makepkg：
+
[source,console]
----
# Clone the AUR package
git clone https://aur.archlinux.org/kwctl.git
cd kwctl

# Build and install the package
makepkg -si
----
+
.. 验证安装：
+
[source,console]
----
kwctl --version
----

. 手动安装
+
.. 下载适用于 Linux 的 `kwctl` 的最新版本：
+
[source,console]
----
curl -LO https://github.com/kubewarden/kwctl/releases/latest/download/kwctl-linux-x86_64.zip
----
+
对于 ARM64 系统（例如 Raspberry Pi），请使用：
+
[source,console]
----
curl -LO https://github.com/kubewarden/kwctl/releases/latest/download/kwctl-linux-aarch64.zip
----
+
.. 从下载的 `.zip` 文件中提取文件：
+
[source,console]
----
unzip kwctl-linux-x86_64.zip
----
+
这将提取以下文件：
+
* `kwctl-linux-x86_64`：`kwctl` 二进制文件
* `kwctl-linux-x86_64.sig`：用于验证二进制文件的签名文件
* `kwctl-linux-x86_64.pem`：用于验证签名的证书文件
+
.. 将二进制文件移动到包含在您的 PATH 中的目录中，重命名为 `kwctl` 并赋予可执行权限。
+
.. 验证安装：
+
[source,console]
----
kwctl --version
----

Windows::
+
--
在 Windows 上安装

. 下载 `kwctl`：
    .. 打开您的浏览器并访问 https://github.com/kubewarden/kwctl/releases/latest[{short-project-name} 发布页面]。
    .. 下载 `kwctl-windows-x86_64.zip` 文件。

. 从下载的 zip 文件中提取文件。它将包含：
+
* `kwctl-windows-x86_64.exe`：`kwctl` 二进制文件。
* `kwctl-windows-x86_64.sig`：用于验证二进制文件的签名文件。
* `kwctl-windows-x86_64.pem`：用于验证签名的证书文件。

. 将二进制文件从 `kwctl-windows-x86_64.exe` 重命名为 `kwctl.exe` 以便于使用。

. 将二进制文件移动到由您的 `PATH` 环境变量覆盖的位置。

. 验证安装。打开新的命令提示符或 PowerShell 窗口，然后检查 `kwctl` 的安装情况：
+
[source,console]
----
kwctl --version
----

== 安装外壳补全

`kwctl` CLI 提供 `--shell` 选项，可以为您的外壳生成外壳补全命令。您可以使用此命令的输出将补全集成到您的外壳中。

[source,console]
----
kwctl completions --shell [bash|elvish|fish|powershell|zsh]
----
