## {project-name} 角色

[cols="1,2", options="header"]
|===
|角色 |说明

|**策略用户**
|一个在集群中使用策略的人。他们部署策略并观察结果。他们配置策略设置，但不编写策略内部代码（如果有的话）。

|**运算符**
|在集群中操作 {short-project-name} 的人。安装和维护 {short-project-name}。

|**策略分发者**
|一个编写了策略并希望与他人分享的人。他们希望策略被“策略用户”角色使用。{short-project-name} 策略的分发者，分发到如 ArtifactHub 或私有 OCI 仓库等地方。

|**策略开发者**
|为他人设计和编写 {short-project-name} 策略的人。

|**集成器**
|以 {short-project-name} 为基础进行构建、使用或开发。也许是一个自定义的用户界面或生成策略的辅助工具。

|**开发人员**
|一个正在处理 {short-project-name} 产品系列的人。构建 {short-project-name} 本身。
|===
