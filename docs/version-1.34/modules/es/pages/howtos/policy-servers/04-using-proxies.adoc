Los PolicyServers respetan las variables de entorno estándar para gestionar configuraciones de proxy. El servidor de directivas utiliza el proxy configurado al descargar directivas.
Esto incluye operaciones de Sigstore al verificar directivas e imágenes.

Estas variables son:

* `HTTP_PROXY` o `http_proxy`: servidor proxy para solicitudes HTTP
* `HTTPS_PROXY` o `https_proxy`: servidor proxy para solicitudes HTTPS
* `NO_PROXY` o `no_proxy`: lista de hosts separados por comas a excluir del proxy

Consulta xref:reference/CRDs.adoc#_policyserverspec[`PolicyServer.spec.env`] en la referencia CRD para establecer estas variables de entorno.
