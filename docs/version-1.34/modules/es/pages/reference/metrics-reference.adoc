== Servidor de Directivas

El componente del Servidor de Directivas inicializa y ejecuta directivas.
Al recibir solicitudes del servidor de la API de Kubernetes, reenvía la solicitud a la directiva, devolviendo la respuesta proporcionada por la directiva al servidor de la API de Kubernetes.

==== `kubewarden_policy_evaluations_total`

===== Equipaje

|===
| Etiqueta | Descripción

| `policy_name`
| Nombre de la directiva

| `resource_name`
| Nombre del recurso evaluado

| `resource_kind`
| Tipo del recurso evaluado

| `resource_namespace`
| Espacio de nombres del recurso evaluado. No presente si el recurso está en el ámbito del clúster.

| `resource_request_operation`
| Tipo de operación: `CREATE`, `UPDATE`, `DELETE`, `PATCH`, `WATCH`...

| `accepted`
| ¿Se aceptó la solicitud?

| `mutated`
| ¿Se modificó la solicitud?

| `error_code`
| Código de error devuelto por la directiva si se rechaza, si lo hay.
  No está presente si la directiva no proporcionó uno.
|===
