== OpenTelemetry, métricas y dependencias de trazado

En el momento de escribir, el stack de https://opentelemetry.io[OpenTelemetry] sigue mejorando. Sin embargo, aún no es estable, y cambios incompatibles hacia atrás no anunciados todavía ocurren. Los desarrolladores de {short-project-name} se esfuerzan por rastrear los cambios en el stack de OpenTelemetry y ajustarse a ellos. El proyecto se prueba contra un rango funcional conocido del stack de OpenTelemetry, métricas y trazado.
