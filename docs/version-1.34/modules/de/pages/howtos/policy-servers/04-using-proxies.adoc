= Konfigurieren von Policy-Servern mit Proxies

:current-version: {page-origin-branch}

Policy-Server respektieren die Standardumgebungsvariablen für den Umgang mit Proxy-Setups. Der Policy-Server verwendet den konfigurierten Proxy beim Herunterladen von Richtlinien.
Dies umfasst Sigstore-Operationen beim Überprüfen von Richtlinien und Bildern.

Diese Variablen sind:

* `HTTP_PROXY` oder `http_proxy`: Proxy-Server für HTTP-Anfragen
* `HTTPS_PROXY` oder `https_proxy`: Proxy-Server für HTTPS-Anfragen
* `NO_PROXY` oder `no_proxy`: durch Kommas getrennte Liste von Hosts, die vom Proxy ausgeschlossen werden sollen

Verweisen Sie auf xref:reference/CRDs.adoc#_policyserverspec[`PolicyServer.spec.env`] im CRD-Referenzdokument, um diese Umgebungsvariablen festzulegen.
