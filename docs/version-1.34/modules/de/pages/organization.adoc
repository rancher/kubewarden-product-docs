Der Großteil des Materials befindet sich in den vier Abschnitten _Tutorials_, _Erklärungen_, _Anleitungen_ und _Referenz_. Es kann Überschneidungen geben, da sich die Abschnitte nicht gegenseitig ausschließen. Das {short-project-name} Projekt versucht, dem link:https://diataxis.fr/[Diátaxis] Dokumentationsrahmen zu folgen.

== Navigation

Das Inhaltsmenü auf der linken Seite zeigt immer Ihren aktuellen Dokumentstandort an.
Das Abschnittsmenü auf der rechten Seite zeigt die aktuellen Seitenabschnitte an. Es gibt auch eine 'Breadcrumbs'-Funktion oben im Hauptinhaltsbereich jeder Seite.

== Suche

Eine Suchfeldfunktion befindet sich im Kopfbereich jeder Seite.

== Bearbeiten

Ein Link zur Quellseite auf GitHub befindet sich am Ende jeder Seite.

== Weitere Ressourcen

Es werden auch folgende Ressourcen gepflegt:

* link:https://github.com/kubewarden/rfc[RFCs] für das Projekt.
* Eine Reihe von link:https://www.kubewarden.io/blog/[Blogbeiträgen], die sich auf {community-name} Veröffentlichungen beziehen.
* Artikel, die sich mit dem link:https://github.com/kubewarden/community[Community]-Management von {community-name} befassen.
