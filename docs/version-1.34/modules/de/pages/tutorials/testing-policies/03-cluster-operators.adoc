=== Testen Sie die Richtlinie

Jetzt kĂ¶nnen Sie `kwctl` verwenden, um die Erstellung eines Pods zu testen, ohne ein AppArmor-Profil anzugeben:

    | jq
{
  "uid": "1299d386-525b-4032-98ae-1949f69f9cfc",
  "allowed": false,
  "status": {
    "message": "These AppArmor profiles are not allowed: [\"unconfined\"]"
