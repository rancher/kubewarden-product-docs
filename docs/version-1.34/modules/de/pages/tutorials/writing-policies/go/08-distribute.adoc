Es war ein langer und hoffentlich angenehmer Weg durch das Go-Tutorial, herzlichen Glückwunsch, dass Sie es bis hierher geschafft haben 🎉.

Während dieses Tutorials haben Sie die https://github.com/kubewarden/safe-labels-policy[safe-labels-policy] nachgebildet.

Es gibt nichts Besonderes zu tun, wenn es um die Verteilung einer Richtlinie geht.
Wenn Sie dieser Anleitung gefolgt sind, haben Sie Ihre Richtlinie bereits mit der GitHub `release.yml` Aktion veröffentlicht, die im letzten Kapitel definiert ist.

Das Thema der Verteilung von Richtlinien wird im Abschnitt xref:/explanations/distributing-policies.adoc["Richtlinien verteilen"] der Dokumentation von {project-name} ausführlicher behandelt.
