PolicyServersは、プロキシ設定に関する標準環境変数を尊重します。ポリシーサーバーは、ポリシーをダウンロードする際に設定されたプロキシを使用します。
これには、ポリシーとイメージを検証する際のSigstore操作が含まれます。

これらの変数は次のとおりです：

* `HTTP_PROXY`または`http_proxy`：HTTPリクエスト用のプロキシサーバー
* `HTTPS_PROXY`または`https_proxy`：HTTPSリクエスト用のプロキシサーバー
* `NO_PROXY`または`no_proxy`：プロキシから除外するホストのカンマ区切りリスト

これらの環境変数を設定するには、CRDリファレンスのxref:reference/CRDs.adoc#_policyserverspec[`PolicyServer.spec.env`]を参照してください。
