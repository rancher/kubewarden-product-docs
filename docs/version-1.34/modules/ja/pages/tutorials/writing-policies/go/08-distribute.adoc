Goチュートリアルを通じて、長く、そしてきっと楽しい旅路を歩んできました。ここまで辿り着いた皆さん、おめでとうございます🎉。

このチュートリアルでは、https://github.com/kubewarden/safe-labels-policy[safe-labels-policy]を再作成しました。

ポリシーを配布する際に特別なことはありません。
このガイドに従った場合、最後の章で定義されたGitHub `release.yml` Actionを使用して、すでにポリシーを公開しています。

ポリシーの配布に関するトピックは、{project-name}のドキュメントのxref:/explanations/distributing-policies.adoc["ポリシーの配布"]セクションでより詳しく説明されています。
