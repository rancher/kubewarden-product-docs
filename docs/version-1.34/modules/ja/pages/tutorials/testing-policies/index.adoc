== ポリシーのテストのペルソナ

このセクションでは、{project-name}ポリシーのテストに関するトピックを扱います。
ポリシーのテストに関心のある2つのペルソナがあります：

* *ポリシー開発者*：あなたは{short-project-name}ポリシーを作成しており、コードが期待通りに動作することを確認したいと考えています。
* *クラスターオペレーター*：あなたは{short-project-name}ポリシーを見つけ、そのポリシー設定をデプロイする前にテストし、調整したいと考えています。
これらの設定をCI/CDパイプラインやその他の管理タスクでテストし続けたいと考えています。

ドキュメントの次のセクションでは、これらのペルソナがどのようにして{short-project-name}ポリシーをテストできるかを示します。
