{project-name}はプロジェクトのドキュメントを次のように整理します：

* xref:/introduction.adoc[概要]
* xref:/quick-start.adoc[クイックスタート]
* _チュートリアル_—例を通じて学ぶ。
* _説明_—基本的な概念。
* _ハウツー_—{short-project-name}のタスクを完了する。
* _リファレンス_
* 構成—このページ
* xref:/personas.adoc[ペルソナ]—{short-project-name}は誰のためですか？
* xref:/disclosure.adoc[セキュリティ]
* xref:/glossary.adoc[用語集]

ほとんどの資料は、_チュートリアル_、_説明_、_ハウツー_、および_リファレンス_の4つのセクションにあります。セクションは相互に排他的ではないため、重複がある場合があります。{short-project-name}プロジェクトは、link:https://diataxis.fr/[ディアタキシス]ドキュメントフレームワークに従うことを試みています。

== ナビゲーション

左側の目次メニューは、常に現在のドキュメントの位置を示します。
右側のセクションメニューは、現在のページのセクションを表示します。各ページのメインコンテンツパネルの上部には、'パンくずリスト'機能もあります。
