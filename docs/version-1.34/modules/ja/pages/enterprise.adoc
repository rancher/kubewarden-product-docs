ifeval::["{build-type}" == "community"]
これらの組織は、{project-name}のエンタープライズサポートを提供します：

[cols="1,1"]
|===
| 組織 | 提供

| link:https://suse.com[SUSE]
| SUSEは、CNCF {kw-community-name}の完全なエンタープライズサポートをlink:https://apps.rancher.io/applications/suse-security-admission-controller[SUSE Security Admission Controller]を通じて提供します。これは、{kw-community-name}SUSE Linux BCIベースのイメージの上に構築された、キュレーションされた信頼できるバージョンです。このソリューションは、エンタープライズグレードのKubernetes環境に対して、強化されたセキュリティ、安定性、および互換性を確保します。
|===
endif::[]
