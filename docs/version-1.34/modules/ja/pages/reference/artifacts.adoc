== OCIアーティファクト

Open Container Initiative (OCI) アーティファクトについて、プロジェクトは、`ghcr.io` 内の https://github.com/orgs/kubewarden/packages. 配下にすべてを公開しています。

- https://github.com/orgs/kubewarden/packages/policies[`kubewarden/packages/policies/*`]はポリシーWasm OCIアーティファクトです。
- https://github.com/orgs/kubewarden/packages/charts[`kubewarden/packages/charts/*`]はHelmチャートとしてのOCIアーティファクトです。SLSA基準に準拠しています。
- OCIイメージとして公開されたコンテナイメージ: `kubewarden-controller`, `policy-server`, `audit-scanner`, `kubectl`。
- e2eテストで使用されるアーティファクトを含む`packages/tests/*`もあります。

== HTTPS経由のHelmチャート

{short-project-name}は、https://charts.kubewarden.io.のHelmチャートHTTPSリポジトリにHelmチャートを公開します。SLSAや署名はありませんが、このタイプのHelmリポジトリは検索機能を提供します。
