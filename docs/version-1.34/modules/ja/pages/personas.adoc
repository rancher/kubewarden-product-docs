## {project-name} ペルソナ

[cols="1,2", options="header"]
|===
|ペルソナ |説明

|**ポリシー利用者**
|ポリシーを取得し、クラスターで使用する人です。彼らはポリシーをデプロイし、結果を観察します。彼らはポリシー設定を構成しますが、ポリシー内部コード（もしあれば）を書くことはありません。

|**演算子**
|クラスター内で {short-project-name} を操作する人です。{short-project-name} をインストールし、維持管理します。

|**ポリシー配布者**
|ポリシーを書いた人で、他の人と共有したい人です。彼らは「ポリシー利用者」役割によってポリシーが利用されることを望んでいます。{short-project-name} ポリシーの配布者で、ArtifactHub やプライベート OCI リポジトリなどの場所に配布します。

|**ポリシー開発者**
|他の人が利用できるように {short-project-name} のためにポリシーを設計し、作成する人です。

|**インテグレータ**
|彼らは{short-project-name}を使ったり、利用したり、その上に構築したりします。おそらく、ポリシーを生成するためのカスタム UI またはヘルパーツールです。

|**開発者**
|{short-project-name} 製品セットに取り組んでいる人。{short-project-name} 自体を構築しています。
|===
