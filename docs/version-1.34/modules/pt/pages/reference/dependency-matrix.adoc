== OpenTelemetry, dependências de métricas e rastreamento

No momento da redação, a pilha https://opentelemetry.io[OpenTelemetry] continua melhorando. No entanto, ainda não está estável, e mudanças incompatíveis com versões anteriores, não anunciadas, ainda ocorrem. Os desenvolvedores do {short-project-name} se esforçam para acompanhar as mudanças na pilha OpenTelemetry e se ajustar a elas. O projeto testa contra um intervalo conhecido de funcionamento da pilha OpenTelemetry, métricas e rastreamento.
