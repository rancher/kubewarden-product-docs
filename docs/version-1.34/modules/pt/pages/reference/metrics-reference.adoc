== Servidor de Políticas

O componente Servidor de Políticas inicializa e executa políticas.
Ao receber solicitações do servidor de API do Kubernetes, ele encaminha a solicitação para a política, retornando a resposta fornecida pela política ao servidor de API do Kubernetes.

==== `kubewarden_policy_evaluations_total`

===== Baggage

|===
| Etiqueta | Descrição

| `policy_name`
| Nome da política

| `resource_name`
| Nome do recurso avaliado

| `resource_kind`
| Tipo do recurso avaliado

| `resource_namespace`
| namespace do recurso avaliado. Não presente se o recurso for de escopo de cluster.

| `resource_request_operation`
| Tipo de operação: `CREATE`, `UPDATE`, `DELETE`, `PATCH`, `WATCH`...

| `accepted`
| A solicitação foi aceita?

| `mutated`
| A solicitação foi modificada?

| `error_code`
| Código de erro retornado pela política se rejeitada, se houver.
  Não está presente se a política não fornecer uma.
|===
