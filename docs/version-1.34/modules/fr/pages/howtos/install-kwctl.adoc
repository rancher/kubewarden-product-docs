= Installation de kwctl ({project-name} CLI)
:revdate: 2026-03-18
:page-revdate: {revdate}
:title: Installation de kwctl
:description: Installez kwctl (le {project-name} CLI) en suivant ses instructions d'installation pour divers systèmes d'exploitation, puis activez la complétion de shell.
:keywords: ["{project-name}", kubewarden, kubernetes, installer kwctl, installer, kwctl]
:doc-persona: kubewarden-all
:doc-type: howto
:doc-topic: install-kwctl

`kwctl` est l'outil d'interface de ligne de commande (CLI) pour {project-name}. Voici les instructions d'installation pour certains systèmes d'exploitation.

[tabs]
====
Linux::
+
--
Installer pour Linux

. Homebrew
+
Si vous utilisez le gestionnaire de paquets Homebrew sur Linux, alors :
+
[source,console]
----
brew install kwctl
----
+
Vérifiez l'installation :
+
[source,console]
----
kwctl --version
----

. Zypper (openSUSE)
+
[source,console]
----
zypper install kwctl
----

. AUR
+
Si vous utilisez Arch Linux ou une distribution basée sur Arch, vous pouvez installer kwctl depuis l'AUR.
+
.. Utilisation d'un helper AUR (yay) :
+
[source,console]
----
yay -S kwctl
----
+
.. Utilisation de makepkg :
+
[source,console]
----
# Clone the AUR package
git clone https://aur.archlinux.org/kwctl.git
cd kwctl

# Build and install the package
makepkg -si
----
+
.. Vérifiez l'installation :
+
[source,console]
----
kwctl --version
----

. Installation manuelle
+
.. Téléchargez la dernière version de `kwctl` pour Linux :
+
[source,console]
----
curl -LO https://github.com/kubewarden/kwctl/releases/latest/download/kwctl-linux-x86_64.zip
----
+
Pour les systèmes ARM64 (par exemple, Raspberry Pi), utilisez :
+
[source,console]
----
curl -LO https://github.com/kubewarden/kwctl/releases/latest/download/kwctl-linux-aarch64.zip
----
+
.. Extrayez les fichiers du fichier `.zip` téléchargé :
+
[source,console]
----
unzip kwctl-linux-x86_64.zip
----
+
Cela extrait les fichiers suivants :
+
* `kwctl-linux-x86_64` : Le binaire `kwctl`
* `kwctl-linux-x86_64.sig` : Un fichier de signature pour vérifier le binaire
* `kwctl-linux-x86_64.pem` : Un fichier de certificat pour vérifier la signature
+
.. Déplacez le binaire dans un répertoire de votre PATH, renommez-le en `kwctl` et rendez-le exécutable.
+
.. Vérifiez l'installation :
+
[source,console]
----
kwctl --version
----

Windows::
+
--
Installez pour Windows

. Téléchargez `kwctl` :
    .. Ouvrez votre navigateur et allez sur la page des versions https://github.com/kubewarden/kwctl/releases/latest[{short-project-name}].
    .. Téléchargez le fichier `kwctl-windows-x86_64.zip`.

. Extrayez les fichiers du fichier zip téléchargé. Il contiendra :
+
* `kwctl-windows-x86_64.exe` : le binaire `kwctl`.
* `kwctl-windows-x86_64.sig` : un fichier de signature pour vérifier le binaire.
* `kwctl-windows-x86_64.pem` : un fichier de certificat pour vérifier la signature.

. Renommez le fichier binaire de `kwctl-windows-x86_64.exe` à `kwctl.exe` pour une utilisation plus facile.

. Déplacez le binaire vers un emplacement couvert par votre variable d'environnement `PATH`.

. Vérifiez l'installation. Ouvrez une nouvelle invite de commandes ou une fenêtre PowerShell et vérifiez l'installation de `kwctl` :
+
[source,console]
----
kwctl --version
----

== Installez les complétions de shell

Le `kwctl` CLI dispose de l'option `--shell` pour générer des commandes de complétion de shell pour votre shell. Vous pouvez utiliser la sortie de cette commande pour intégrer des complétions dans votre shell.

[source,console]
----
kwctl completions --shell [bash|elvish|fish|powershell|zsh]
----
